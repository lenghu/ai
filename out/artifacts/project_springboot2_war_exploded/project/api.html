请求方式： GET

请求参数说明

序号 参数名称 必选 类型 描述
1 url 是 String 查询后台数据库中相关url的信息

响应参数说明

序号 参数名称 类型 描述
1 cheatWebsiteJson Json 若数据库中存储了相关url，则cheatWebsite返回该url在数据库中的基本信息,code=200； 若没有，则cheatWebsite返回null，code=300; 若查询失败，则code=400
